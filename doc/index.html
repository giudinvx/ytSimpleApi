Introduction
ytSimpleApi, is a simple plugin that make easy use youtube Api, with this class
you can create a youtube box in your website, infact you can extract same info from a youtube url or id.

Installation
Copy the file ytSimpleApi.php and /zend footer in your website root, or where you want use this.

Implementation
First you must know that this plugin was design for use in two way, can instance an object or
call any method with scope operator "::".
I now show you what you can do with this.

Create a new file and call the class:

require_once( "ytSimpleApi.php" );


Instance it's with an id or url (we use a url):

$instanceyt = new ytSimpleApi("http://www.youtube.com/watch?v=the0KZLEacs");


Function:

info()
Now we can use this instance for extract some info with "info()" function,
default it's return an array but if you want a json code you can.

$array_info = $instanceyt->info();
print_r( $array_info );

comment()

 Extract comment function description:
This is really useful function because you can use this for extract all comment,
only n comment or comment between n and n:

Extract all comment:

$instanceyt->comment();


Extract only first 5 comment:

$instanceyt->comment(5);


Extract comment between 5 and 10:

$instanceyt->comment(5,10);

First argument where you want start, and second argument where you want came.

embed()

With this function your embed a video in your site:
You can call this function with two argument, first width and second height,
if you don't set argument default are: width 640 and height 385.

Set width and height:

$instanceyt->embed(400,150);


Set only width:

$instanceyt->embed(400);


Set only height:

$instanceyt->embed(null,150);


Use Scope Resolution Operator (::)

As I mentioned above, you can use this class also with this operator,
remember you'll always set a url or id in all function, there some example:

echo ytSimpleApi::embed("http://www.youtube.com/watch?v=the0KZLEacs");


Extract comment

ytSimpleApi::comment("the0KZLEacs"); //remember you can use also only a valid id


Extract some info

print_r(ytSimpleApi::info("the0KZLEacs"));
